## 팀원 소개 및 역할 분담

- 공통 사항
    - 데이터베이스 설계 및 SPRING BOOT REST API 구현
    - axios 기반 프론트엔드 - 백엔드 데이터 통신 기능 구현
- 김은지
    - CSS 레이아웃 디자인 설계 및 구현
    - 유저, 게시글, 댓글 관련 CRUD 기능 구현
- 조한빈
    - CSS 레이아웃 디자인 구현
    - 게시글, 장소 좋아요(찜) 기능 구현
    - 유저 간 팔로잉 기능 구현
