    - CSS 레이아웃 디자인 설계 및 구현
    - 유저, 게시글, 댓글 관련 CRUD 기능 구현
- 조한빈
    - CSS 레이아웃 디자인 구현
    - 게시글, 장소 좋아요(찜) 기능 구현
    - 유저 간 팔로잉 기능 구현
